Mp3exe mp3 player.

  Version 2.4, April 2009
  Author: Zoran Cindori
  Email: zcindori@inet.hr
  Web: http://www.inet.hr/~zcindori/mp3exe/


This is simple MP3 player based on LIBWMP3 library.
Web: http://www.inet.hr/~zcindori/libwmp3/

Features:
- support Layer1, Layer2 and Layer3.
- REVERSE PLAYING ( you can play mp3 backward in real time, "real" backward playing without predecoding in PCM)
- support free format bitstream ( e.g. bitrate over 320 kbps )
- read ID3v1 and ID3v2 tags
- built in external Shibatch Super Equalizer ver 0.03 ( 10 band )
- built in echo and reverb sound processor
- VU meter
- advanced FFT spectral graph
- convert mp3 file to self playing exe


I made this simple mp3 player to demostrate usage of LIBWMP3 multimedia library.
